Movie Matching

Home Projects

Description

Movie Matching is a web application that helps users discover movies and TV shows. Users can receive personalized recommendations based on their favorite genres and save their favorite titles to their account. The project leverages the TMDB API to fetch up-to-date movie and series data.

Features

  • Trending and Now Playing: Browse popular and currently playing movies.
  • Personalized Recommendations: Get movie suggestions based on selected genres.
  • Search Functionality: Search for movies and TV shows using an integrated search bar.
  • User Account Management: Users can register, log in, and manage their profiles.
  • Favorites: Users can save and manage a list of their favorite movies.
  • Profile Picture Uploads: Users can upload and customize their profile pictures.

Technologies Used

  • Frontend: HTML, CSS, EJS (Embedded JavaScript Templates)
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: Bcrypt for password hashing and Express-session for session management
  • API: TMDB API for movie and series data
  • File Handling: Multer for profile picture uploads

Key Learning Outcomes

  • Working with external APIs (TMDB) and processing JSON data.
  • Implementing user authentication and session management.
  • Using MongoDB to store user data and favorites.
  • Dynamically rendering pages with EJS.
  • Handling file uploads and securely storing them.
