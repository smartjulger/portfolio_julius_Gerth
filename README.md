# Julius Gerth's Portfolio

This is the source code for my personal portfolio website. It showcases my projects and skills as a web developer.

## Description

The portfolio is a static website built with HTML, CSS, and vanilla JavaScript. It is designed to be a clean and simple showcase of my work. The site is fully responsive and works well on all devices.

## Setup

To run this project locally, you can simply open the `index.html` file in your web browser. There are no special dependencies or build steps required.

## Technologies Used

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For styling and layout.
*   **JavaScript:** For interactive elements and functionality.
*   **Git:** For version control.
*   **GitHub Pages:** For hosting the live website.
